Installation & System Setup

ইন্সটলেশন ও সিস্টেম সেটআপ
🖥️ Windows Requirements
  • ✓
    OS: Windows 10 (64-bit) বা তার উপরে
  • ✓
    RAM: ন্যূনতম 16GB (32GB রেকমেন্ডেড)
  • ✓
    GPU: 4GB VRAM সহ GPU
  • ✓
    Storage: 15GB ফ্রি স্পেস (SSD হলে ভালো)
  • ✓
    CPU: Intel Core i7 / AMD Ryzen 7+
🍎 Mac Requirements
  • ✓
    OS: macOS 12.0 Monterey বা উপরে
  • ✓
    RAM: 16GB (Apple Silicon-এ 8GB চলে)
  • ✓
    CPU: Apple M1/M2/M3 বা Intel Core i7
  • ✓
    Storage: 15GB ফ্রি স্পেস
  • ✓
    Display: 1280×1024 রেজোলিউশন
📥 ইন্সটলেশনের ধাপগুলো
  • ১
    Adobe Creative Cloud ডাউনলোড করুন: creativecloud.adobe.com থেকে
  • ২
    Adobe ID দিয়ে লগইন করুন অথবা নতুন অ্যাকাউন্ট তৈরি করুন
  • ৩
    Creative Cloud App খুলে "After Effects" খুঁজুন এবং Install বাটনে ক্লিক করুন
  • ৪
    ইন্সটলেশন সম্পন্ন হলে Launch বাটনে ক্লিক করে চালু করুন
  • ৫
    প্রথমবার চালু হলে Preferences সেটআপ করুন (Edit → Preferences → General)
⚠️
সতর্কতা
After Effects চালানোর সময় অন্য ভারী সফটওয়্যার বন্ধ রাখুন। RAM কম থাকলে প্রোগ্রাম ক্র্যাশ করতে পারে।
⚙️ প্রথমেই করবেন এই সেটিংস
  • ১
    Disk Cache: Edit → Preferences → Media & Disk Cache — ক্যাশ ফোল্ডার অন্য ড্রাইভে সেট করুন
  • ২
    RAM Allocation: Edit → Preferences → Memory — After Effects-কে বেশি RAM দিন
  • ৩
    Auto-Save: Edit → Preferences → Auto-Save → Enable করুন (প্রতি ৫ মিনিটে)
  • ৪
    GPU Acceleration: File → Project Settings → Video Rendering → Mercury GPU Acceleration

Chapter 03

Creating Compositions

কম্পোজিশন তৈরি করা
📐 Composition কী?

Composition হলো After Effects-এর কাজের ক্ষেত্র (Canvas)। প্রতিটি প্রজেক্টে এক বা একাধিক কম্পোজিশন থাকতে পারে। প্রতিটি কম্পোজিশনে Resolution, Frame Rate, Duration এবং Background সেট করা যায়।

🆕 নতুন Composition তৈরির ধাপ
  • ১
    Composition Menu → New Composition — অথবা Ctrl+N
  • ২
    Preset থেকে রেডি-মেড সাইজ বেছে নিন: HD 1920×1080, 4K 3840×2160, বা Custom
  • ৩
    Frame Rate সেট করুন: YouTube/Social = 30fps, Cinema = 24fps, Gaming = 60fps
  • ৪
    Duration দিন: কতক্ষণের ভিডিও তৈরি করবেন (যেমন: 0:00:10:00 = ১০ সেকেন্ড)
  • ৫
    Background Color বেছে নিন এবং কম্পোজিশনের নাম দিন, তারপর OK চাপুন

🔑 Layer Properties (Transform)

প্রতিটি লেয়ারে Transform অংশে ৫টি মূল প্রপার্টি থাকে। Timeline-এ লেয়ার সিলেক্ট করে P, S, R ইত্যাদি চাপলে আলাদাভাবে দেখা যায়:

Chapter 05

Keyframe Animation

কীফ্রেম অ্যানিমেশনের মূল কৌশল
🔑 Keyframe কী?

Keyframe হলো টাইমলাইনের একটি নির্দিষ্ট বিন্দু যেখানে একটি লেয়ারের প্রপার্টির মান রেকর্ড করা হয়। দুটি Keyframe-এর মাঝে After Effects স্বয়ংক্রিয়ভাবে মসৃণ ট্রানজিশন তৈরি করে — এটাই অ্যানিমেশন।

🎬 প্রথম অ্যানিমেশন তৈরি (ধাপে ধাপে)
  • ১
    Timeline-এ লেয়ার সিলেক্ট করুন এবং P চেপে Position দেখান
  • ২
    টাইমইন্ডিকেটর 0:00 সেকেন্ডে রাখুন। Position-এর পাশের স্টপওয়াচ আইকন ক্লিক করুন — Keyframe রেকর্ডিং চালু হবে
  • ৩
    এখন অবজেক্টের শুরুর অবস্থান সেট করুন (যেমন: স্ক্রিনের বাইরে বাম দিকে)
  • ৪
    টাইমইন্ডিকেটর 1:00 (১ সেকেন্ড) এ নিয়ে যান
  • ৫
    অবজেক্টকে কাঙ্ক্ষিত অবস্থানে নিয়ে যান — স্বয়ংক্রিয়ভাবে নতুন Keyframe তৈরি হবে
  • ৬
    Space চেপে প্রিভিউ দেখুন — অবজেক্ট বাম থেকে ডানে মুভ করবে!

Chapter 06

Effects & Presets

ইফেক্ট ও প্রিসেটের ব্যবহার
✨ ইফেক্ট যোগ করার উপায়
  • ১
    Timeline-এ লেয়ার সিলেক্ট করুন
  • ২
    Effects & Presets Panel-এ সার্চ করুন (যেমন: "blur", "glow", "color")
  • ৩
    ইফেক্টটি লেয়ারের উপর ড্র্যাগ করুন অথবা ডাবল-ক্লিক করুন
  • ৪
    Effect Controls Panel-এ ইফেক্টের সেটিংস সামঞ্জস্য করুন
🌟 সবচেয়ে কাজের ইফেক্টগুলো
🌊
Gaussian Blur
মসৃণ ব্লার ইফেক্ট। Blurriness বাড়িয়ে কমিয়ে ডেপথ তৈরি করুন।
✨
Glow
আলোকিত গ্লো ইফেক্ট। নিয়ন লাইট ও সাই-ফাই লুকের জন্য।
🎨
Hue/Saturation
রঙ পরিবর্তন। Colorize অপশনে সম্পূর্ণ রঙ বদলে ফেলুন।
🌈
Curves
প্রফেশনাল কালার গ্রেডিং। RGB চ্যানেল আলাদা নিয়ন্ত্রণ।
⚡
CC Particle World
পার্টিকেল সিস্টেম। আগুন, ধোঁয়া, তারার ইফেক্ট তৈরি করুন।
💥
Stroke
পাথ ড্র করে লাইন অ্যানিমেট করুন। লোগো অ্যানিমেশনে দারুণ।
🔲
Drop Shadow
ছায়া যোগ করুন। Depth ও Dimension তৈরি করে।
🌀
Turbulent Displace
ঢেউ বা বিকৃতি ইফেক্ট। জলের প্রতিফলন তৈরিতে ব্যবহার হয়।
Chapter 07

Text Animation

টেক্সট অ্যানিমেশন তৈরি করা
✍️ টেক্সট লেয়ার তৈরি করুন
  • ১
    Toolbar থেকে T (Text Tool) বেছে নিন অথবা Ctrl+T
  • ২
    Composition Panel-এ ক্লিক করে টাইপ করুন
  • ৩
    Character Panel থেকে Font, Size, Color সেট করুন
  • ৪
    Paragraph Panel থেকে Alignment (Left/Center/Right) সেট করুন
  • ৫
    Selection Tool (V) দিয়ে টেক্সট সরান বা রিসাইজ করুন
🎭 Animator দিয়ে টেক্সট অ্যানিমেশন

Text Layer → Animate বাটন ক্লিক করলে বিভিন্ন অ্যানিমেশন প্রপার্টি পাওয়া যায়। Animator ব্যবহার করে প্রতিটি অক্ষর আলাদাভাবে অ্যানিমেট করা যায়।

  • ১
    Text Layer সিলেক্ট করুন → Animate → Opacity যোগ করুন
  • ২
    Range Selector → Offset অ্যানিমেট করুন: 0% থেকে 100%
  • ৩
    Advanced → Shape: Ramp Up বেছে নিন
  • ৪
    এখন প্রতিটি অক্ষর একে একে ফেড ইন হবে!

Chapter 09

Expressions

এক্সপ্রেশন — অ্যানিমেশনের সুপারপাওয়ার
💻 Expression কী?

Expression হলো JavaScript-ভিত্তিক কোড যা After Effects-এ প্রপার্টির মান স্বয়ংক্রিয়ভাবে নিয়ন্ত্রণ করে। Keyframe ছাড়াই জটিল অ্যানিমেশন তৈরি করা সম্ভব।

Expression যোগ করুন: Alt + প্রপার্টির স্টপওয়াচ আইকনে ক্লিক করুন।

🔥 সবচেয়ে দরকারি Expressions
  • ১
    wiggle() — এলোমেলো কাঁপুনি ইফেক্ট
    wiggle(5, 30) — প্রতি সেকেন্ডে ৫ বার, ৩০ পিক্সেল পর্যন্ত
  • ২
    time — সময়ের সাথে ঘূর্ণন
    time * 90 — প্রতি সেকেন্ডে ৯০° ঘোরে
  • ৩
    loopOut() — অ্যানিমেশন লুপ করুন
    loopOut("cycle") — অনন্তকাল লুপ হতে থাকবে
  • ৪
    random() — র‍্যান্ডম মান
    random(0, 100) — ০ থেকে ১০০-এর মধ্যে র‍্যান্ডম
  • ৫
    thisComp.layer().transform.position — এক লেয়ার আরেক লেয়ারকে অনুসরণ করবে (Pick Whip)
⚠️
মনে রাখুন
Expression সঠিকভাবে না লিখলে হলুদ রঙের Error দেখাবে। Expression ডিসেবল করতে = আইকনে ক্লিক করুন।
